# 🥗 NutriScan

**NutriScan** is an AI-powered nutrition analysis application that helps users understand the nutritional value of their meals from a simple food image. Using **Google Gemini 2.5 Flash**, the application analyzes uploaded food images and answers nutrition- and diet-related questions.

## 🚀 Features

* 📷 Upload food images (JPG, JPEG, PNG, AVIF)
* 🤖 AI-powered food analysis using Google Gemini 2.5 Flash
* 🥦 Nutrition-focused responses
* 🔥 Calorie estimation
* 💪 Protein, carbohydrate, fat, and fiber insights
* 🩺 Healthy diet recommendations
* ⚡ Fast and interactive user experience
* 🎨 Clean and responsive interface
